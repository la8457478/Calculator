# Stock & Futures Trading Calculator

一个专业的股票和期货交易计算工具，帮助您进行仓位管理和止盈止损计算。

🔗 **在线体验**：部署后访问 `https://你的GitHub用户名.github.io/Calculator/`

## 功能特性

### 🔄 双模式支持
- **股票模式**：适用于股票交易的仓位和止盈计算
- **期货模式**：支持62种期货品种，涵盖贵金属、黑色系、有色金属、化工、能源、农产品、新能源等八大类

### 📊 仓位计算器
- 输入买入价/开仓价、止损价和最大风险金额
- **股票模式**：自动计算应购买股数、总投资金额和实际风险
- **期货模式**：计算建议开仓手数、预估保证金和每手风险

### 🎯 止盈计算器
- 基于1:1风险回报比
- 自动计算止盈目标价
- **期货模式**：显示每手风险（点数和金额）
- 可视化价格区间展示

### 🔍 模糊搜索
- 支持中文名称或代码快速筛选期货品种
- 分类清晰，包含八大类别

### 📦 内置期货品种（62种）
- **贵金属**：黄金、白银、铂、钯
- **黑色系**：螺纹钢、热轧卷板、铁矿石、焦炭、焦煤、不锈钢、线材、硅铁、锰硅
- **有色金属**：铜、铝、锌、铅、镍、锡、氧化铝、铸造铝合金、国际铜
- **化工**：天然橡胶、纸浆、PVC、玻璃、纯碱、甲醇、PTA、乙二醇、尿素等18种
- **能源**：原油、燃料油、低硫燃料油、石油沥青
- **农产品**：豆粕、玉米、白糖、棉花、生猪、鸡蛋、苹果、红枣等20种
- **新能源**：碳酸锂、工业硅、多晶硅
- **其他**：集运欧线指数

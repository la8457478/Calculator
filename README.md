# Stock & Futures Trading Calculator

一个专业的股票和期货交易计算工具，帮助您进行仓位管理和止盈止损计算。

🔗 **在线体验**：[访问计算器](https://yourusername.github.io/stock-calculator/)

## 功能特性

### 🔄 双模式支持
- **股票模式**：适用于股票交易的仓位和止盈计算
- **期货模式**：支持17种期货品种，包括贵金属、黑色系、有色金属、化工、能源和农产品

### 📊 仓位计算器
- 输入买入价/开仓价、止损价和最大风险金额
- **股票模式**：自动计算应购买股数、总投资金额和实际风险
- **期货模式**：计算建议开仓手数、预估保证金和每手风险

### 🎯 止盈计算器
- 基于1:1风险回报比
- 自动计算止盈目标价
- **期货模式**：显示每手风险（点数和金额）
- 可视化价格区间展示

### 📦 内置期货品种
- **贵金属**：黄金(AU)、白银(AG)
- **黑色系**：螺纹钢(RB)、热轧卷板(HC)、铁矿石(I)、焦炭(J)、焦煤(JM)
- **有色金属**：铜(CU)、铝(AL)、锌(ZN)
- **化工**：PTA(TA)、甲醇(MA)、聚丙烯(PP)
- **能源**：原油(SC)、燃料油(FU)
- **农产品**：豆粕(M)、豆油(Y)、棕榈油(P)、玉米(C)

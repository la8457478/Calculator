# Stock Trading Calculator

一个专业的股票交易计算工具，帮助您进行仓位管理和止盈止损计算。

🔗 **在线体验**：[访问计算器](https://yourusername.github.io/stock-calculator/)

## 功能特性

### 📊 仓位计算器
- 输入买入价、止损价和最大风险金额
- 自动计算应购买股数
- 显示总投资金额和实际风险

### 🎯 止盈计算器
- 基于1:1风险回报比
- 自动计算止盈目标价
- 可视化价格区间展示

## 快速开始

直接访问在线版本，或下载 `index.html` 文件本地使用。

## 技术栈

- Pure HTML/CSS/JavaScript
- 响应式设计
- 无需任何依赖
